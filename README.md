# Wished

A Disney World trip planning tool.

## Deploy to Vercel (no Terminal needed)

1. Sign up at github.com (free)
2. Create a new repository called "wished" (public)
3. Upload all files in this folder to the repository
4. Sign up at vercel.com using your GitHub account
5. On the Vercel dashboard, click "Add New" → "Project"
6. Select your "wished" repository, click "Import"
7. Click "Deploy" — accept all defaults
8. Wait 1–2 minutes. You'll get a live URL.

That's it. Share the URL.
